Example

Usage example:

var aa = new MLJ.util.AssociativeArray();
aa.set("key1", obj1);
aa.set("key2", obj2);
var iter = aa.iterator();
var obj;
while(iter.hasNext()) {
   obj = iter.next();
   //do something with obj
}
